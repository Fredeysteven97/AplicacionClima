# AplicacionClima

Aplicación web del clima en tiempo real con detección automática de ubicación. Desarrollada con tecnologías web estándar sin dependencias externas.

## Funcionalidades

- Detección automática de ubicación al cargar la app
- Búsqueda de clima por nombre de ciudad
- Temperatura actual en grados Celsius
- Descripción del clima en español
- Humedad, velocidad del viento y visibilidad
- Fondo dinámico que cambia según la condición climática
- Manejo de errores para ciudades no encontradas

## API

Este proyecto utiliza la API de [OpenWeatherMap](https://openweathermap.org/) para obtener los datos del clima en tiempo real.

Para ejecutarlo localmente necesitas tu propia API key gratuita en [openweathermap.org](https://openweathermap.org/api).

---

## Conceptos aplicados

- Consumo de APIs externas con fetch y async/await
- Geolocation API del navegador
- Manipulación del DOM con JavaScript puro
- Manejo de errores con try/catch
- Diseño responsive con CSS Grid y Flexbox
- Glassmorphism y gradientes dinámicos con CSS
- Variables CSS para gestión de paleta de colores

## Instalación

No requiere instalación. Accede directamente desde el navegador:

[https://fredeysteven97.github.io/AplicacionClima](https://fredeysteven97.github.io/AplicacionClima)

O clona el repositorio:

```bash
git clone https://github.com/Fredeysteven97/AplicacionClima.git
```

Reemplaza `TU_API_KEY_AQUI` en `script.js` con tu API key de OpenWeatherMap.
